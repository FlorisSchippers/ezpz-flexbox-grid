Default features

This page contains examples of the grid features that are enabled by default.

All examples on this page are based on a 12 column grid system. Please refer to the examples of the fraction based grid system if you prefer this over a column based setup.

4: Cells with specific sizes

By adding a .cell--{#} class to a .cell element a percentual width can be applied to the cell. A .cell--1 class means that the element will take up 1 cell out of a maximum of 12. An element with the .cell--6 class will take up 6 cell spaces etc.
