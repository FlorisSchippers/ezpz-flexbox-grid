4: Cells with specific sizes

There are two ways to force a certain size upon a cell, depending on the chosen grid type. This can either be 'columns' or 'fractions'. Different classes are used to achieve the desired width of the cell(s) for these options.

4.1: When using columns

By adding a .cell--{#} class to a .cell element a percentual width can be applied to the cell. A .cell--1 class means that the element will take up 1 cell out of a maximum of 12. An element with the .cell--6 class will take up 6 cell spaces etc.

4.2: When using fractions

When the grid style is set to 'fractions' cell width can be defined by adding a .cell--{fraction} class to a .cell element.

A .cell--2/3 class means that the element will take up two-thirds of the available space in the grid section. An element with the .cell--1/2 class will take up one-half of the grid section, etc.

By only applying a .cell--{fraction} to specific cells, a width can be set for these cells. The remaining cells will equaly divide the remaining space within the grid section automatically.
